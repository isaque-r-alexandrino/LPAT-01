**Calculadora em Python**
Uma calculadora simples e funcional desenvolvida em Python, com interface via linha de comando.

**Funcionalidades**
Esta calculadora oferece as seguintes operações matemáticas:

-  **Soma**: Adição de dois números
-  **Subtração**: Subtração de dois números
-  **Multiplicação**: Multiplicação de dois números
-  **Divisão**: Divisão de dois números (com tratamento para divisão por zero)
-  **Potência**: Cálculo de potência (base elevada ao expoente)
-  **Raiz Quadrada**: Cálculo da raiz quadrada de um número (com tratamento para números negativos)

**Como Usar**

**Pre-requisitos**
-  Python 3.x instalado em seu sistema

**Execução**

1. Clone ou baixe o arquivo
2. Navegue até o diretório do arquivo no terminal
3. Execute o comando
